# GIALLOVISION // CORE

**Interactive Nodal Portfolio Interface** A high-performance, browser-based experiential portfolio built to mimic the nodal workflow environments of professional media servers, AI generation pipelines, and system engineering architectures.

![Giallovision Interface](assets/images/TouchOSC.png) ## ⚙️ Core Architecture

This application bypasses standard web design conventions to deliver a fully interactive, node-based workspace. Visitors physically interact with the UI by routing virtual cables, triggering events, and exploring system states.

* **Frontend Engine:** Vanilla HTML5, CSS3, ES6 JavaScript
* **Nodal Framework:** [LiteGraph.js](https://github.com/jagenjo/litegraph.js) (Customized)
* **Background Rendering:** Custom WebGL GLSL Shaders (Pixel-perfect procedural noise and fluid sims)
* **Hosting / CI:** GitHub Pages (Linux) + Cloudflare SSL
* **Contact Database:** Headless Google Forms "Stealth Bridge" via hidden iframe targeting

## 🖥️ System Modules (Nodes)

* **Console / Boot Sequence:** Simulates a Linux kernel boot sequence, establishes the overarching pipeline, and dynamically loads initial connection states.
* **Exosphere & Canopy Render:** Representing large-format LED mapping architectures.
* **KSampler & VAE Decode:** Simulates a generative AI pipeline.
* **Touchdesigner / OSC Input:** Generates randomized Sample & Hold control data to visually represent live show modulation.
* **Video Combine (VHS):** The primary output canvas. Asynchronously fetches and plays high-resolution video and image assets seamlessly within the nodal graph.
* **Clear Comms:** A fully functional, zero-API, spam-shielded contact module that routes messages directly to a secure Google database without exposing raw email addresses to web scrapers.
